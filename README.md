# Proyecto-M-dulo-6-ABP
Investigación estadística completa sobre los factores que influyen en los hábitos saludables (sueño, alimentación y actividad física) de jóvenes universitarios. El estudio fue conducido aplicando metodologías rigurosas del método científico, análisis probabilístico, distribuciones estadísticas, teoría de muestreo e inferencia estadística.
